# What Is a Memory Capsule?

## One-Liner

A Memory Capsule is a standardised digital package for AI memory — like a shipping container for your conversations, preferences, and project context — making it portable across any AI platform.

## Context

Imagine switching from one AI platform to another. You have spent months — maybe years — building a relationship. The AI knows your preferences, your project history, your communication style. It understands your half-finished sentences.

Now you want to move. Can you take any of that with you?

Usually, no. Your memory is locked inside the platform. Export? Not available. Migration? Not possible. Your digital past stays behind.

This is not a technical limitation. It is a format problem. No one defined what AI memory should look like — so every platform stores it differently, and none of them can read each other's formats.

## Insight

A **Memory Capsule [IR-CAP-006]** solves this.

It is a standardised digital package for AI memory. Think of it like a shipping container. Regardless of what is inside — toys, electronics, furniture — the container规格 is the same everywhere. Ships, ports, and trucks all work with the same standard. No one reinvents the wheel for different cargo.

A Memory Capsule does the same for AI memory. It packages your conversation history, preferences, and project context into a common format that any MCP-compliant platform can read, understand, and continue.

A qualified capsule meets three requirements:

- **Content readable** — any platform can parse it
- **Privacy可控** — client-side encrypted, you hold the key
- **Source traceable** — signatures verify authenticity and integrity

These are not technical details. They are user rights, encoded into the data itself.

## Implication

When Memory Capsules become the norm, users are no longer hostages. You can switch platforms without losing your digital past. Your memory becomes yours — portable, migratable, and never held captive.

For platforms, the competition shifts from locking users in to earning their stay. For the industry, it turns AI from a collection of walled gardens into an open ecosystem.

The capsule does not replace the AI. It carries what the AI learned about you. And that — your memory — is the only thing that truly matters.
