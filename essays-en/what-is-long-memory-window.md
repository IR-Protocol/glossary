# What Is Long Memory Window? — A Chat Page That Never Disappears

## One‑Liner
The Long Memory Window gives you a fixed, permanent conversation entry point where every chat continues where you left off — across devices, across time, with your memory truly owned by you.

## Context
Have you noticed that every AI chat today starts with a blank “new session”? You finish a conversation, close the window, and next time you open it, it is a fresh start. To continue a previous topic, you have to dig through a history list. Switch devices? That list may not even follow you. This is the logic of a tool: tools do not remember people. Every time you pick it up, you start over.

## Insight
The AI Memory Chain whitepaper introduces a different concept: the **Long Memory Window [IR-LMW-011]**. Imagine a fixed, permanent conversation entry point. You open it, and there is no “new session” button — because it is not needed. There is only this one window. Every time you return, you continue exactly where you left off. A project discussion from yesterday is still there. Preferences mentioned last week are still remembered. A standard AI chat is like a hotel — you check in each time, the room is cleared after checkout, and you get a new room number on your next stay. The Long Memory Window is like your own home: the door number never changes, and everything stays where you left it. Every conversation generated inside this window is automatically saved to your encrypted cloud memory space — independent, client‑side encrypted, with the key held only by you. The platform only stores ciphertext and never sees your plaintext. Want to erase everything? One click clears the window and the memory space. Want to take your memory elsewhere? One‑click export, and all conversations migrate intact.

## Implication
This fixed window is not free; it is enabled when you subscribe to the storage‑rent plan. Without storage rent, you only get temporary sessions — new conversations, a history list, use‑and‑forget. Pay the monthly rent, and the door lights up. The window itself is not the novelty; what matters is who owns what is inside. Does it belong to the platform, or to you? Can you take it with you, or is it locked away? The Long Memory Window chooses the latter — a door that stays open for you, with AI that remembers who you are. This is the clearest distinction between “AI as a tool” (always new, leaving no trace) and “AI as a companion” (always there, continuing without re‑introduction).
